Checkbox Group
Checkbox Group creates multiple checkboxes by using list of values or by passing individual checkboxes. It handles all these checkboxes together and selection of one or more checkboxes is possible as a form field. It is implemented to work with Platform Form.

Platform specific enhancements:

  • Checkbox group can create multiple checkboxes by using list of string values.
  • Multiple checkboxes can be created using list of SelectItem Objects.
  • Checkbox group can be created by passing checkboxes under checkbox group.
  • It is implemented to work with Platform Form.
  • Checkbox group creates Platform Checkboxes.
